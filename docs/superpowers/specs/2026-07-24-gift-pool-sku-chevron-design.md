# 赠品池关联 SKU 数下拉图标设计

## 目标

让赠品管理页“赠品池”表格中“关联 SKU 数”列的数字后图标，与产品列表“所属系列”列保持一致。

## 设计

- 将当前 14×14 的圆形信息图标替换为产品列表同款 12×12 向下箭头。
- 复用相同的 SVG 路径、描边宽度、端点和连接样式。
- 保留数字与图标的紧凑横向排列。
- 保留现有悬停高亮和关联 SKU 明细浮层，不改变触发方式、内容或定位逻辑。
- SKU 数为 0 时继续只显示数字 `0`，不显示箭头。

## 实施范围

- 修改 `admin/gift/js/gift_list.js` 中关联 SKU 数单元格的图标标记。
- 如视觉对齐需要，仅微调 `admin/gift/css/gift.css` 中该单元格的间距或悬停选择器。
- 不修改数据结构、SKU 统计、分页、筛选和其他列。

## 验收标准

1. 非零 SKU 数后显示与产品列表“所属系列”一致的 12×12 向下箭头。
2. 图标默认使用弱化颜色，悬停时按现有主题高亮。
3. 悬停图标仍能显示正确的关联 SKU 明细。
4. SKU 数为 0 时不显示箭头。
5. 赠品池列表布局、分页和其他操作无回归。
