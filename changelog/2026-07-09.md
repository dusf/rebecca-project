# 2026-07-09 变更记录

## 1. 店铺下拉面板导航修复

**文件**: `admin/common/commons.js`

- "店铺管理"按钮点击后，通过 `handleSidebarNav('shop-list')` 在 iframe 内加载 `shop/shop_list.html`
- "成员管理"按钮点击后，通过 `handleSidebarNav('members')` 在 iframe 内加载 `shop/members.html`
- 点击这两个按钮时，左侧导航栏保持存在，不再直接跳转替换整个页面

## 2. 面包屑导航修正

**文件**: `admin/index.html`

- `members` 路由面包屑维持 `全局 / 成员管理`

## 3. 成员管理表格新增字段

**文件**: `admin/shop/members.html`

表格新增「账号ID」列，使用等宽字体 `monospace` 展示。

表头最终顺序：复选框 → 成员信息 → **账号ID** → 手机号 → 组织机构 → 所在店铺 → 加入时间 → 操作

## 4. 成员管理修复：移除身份列 & 隐藏账号ID输入框

**文件**: `admin/shop/members.html`

- 移除表格「身份」列（表头 + 数据渲染 + `identityHtml` 生成逻辑 + `.identity-badge` CSS）
- 移除「添加成员」弹窗中的账号ID输入框，账号ID改由 `doAddMember()` 内部调用 `generateAccountId()` 自动生成（格式：`AC年月+5位序号`，如 `AC20260700001`），前端不再展示
- 移除无用 CSS 类 `.account-id-preview`
