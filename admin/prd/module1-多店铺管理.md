# RebeccaShop 商城后台 - 多店铺管理 PRD

## 1. 需求背景与价值

### 1.1 背景
RebeccaShop 是一个对标 Shopify 的跨境电商 SaaS 平台。商家需要用同一账号管理多个品牌或多个区域的店铺，每个店铺独立运营（独立语言、币种、域名），但统一在后台管理。

### 1.2 价值
- **用户价值**：商家无需注册多个账号即可管理多个品牌/区域店铺，降低运营成本
- **业务价值**：提升商家粘性，单商家可创建多店铺提升平台 GMV

## 3. 需求详情

### 3.1 店铺列表页

#### 3.1.1 顶部导航栏
- 入口：登录后默认进入
- 页面结构：顶部显示「RebeccaShop」Logo + 当前店铺名称下拉切换 + 用户头像
- 交互逻辑：
  - 点击店铺名称下拉框，展示所有店铺列表，点击切换
  - 点击用户头像，弹出下拉菜单（账号设置 / 退出登录）
- 异常处理：无店铺时提示「您还没有店铺，点击创建」

#### 3.1.2 店铺卡片列表
- 入口：登录后即见
- 页面结构：
  - 页面标题"我的店铺"
  - 右上角「创建店铺」按钮
  - 卡片网格布局，每张卡片展示：店铺名称、域名、状态标签（启用/停用）、创建时间
  - 每张卡片悬停显示「编辑」「停用/启用」操作按钮
- 交互逻辑：
  - 点击卡片进入该店铺后台
  - 点击「创建店铺」弹出创建对话框
  - 点击「编辑」弹出编辑对话框
  - 点击「停用」二次确认后切换状态
- 异常处理：停用店铺时弹出确认框「停用后用户端将无法访问，确定继续？」

#### 3.1.3 店铺名称下拉切换
- 入口：顶部导航栏左侧
- 页面结构：
  - 店铺列表区：每行显示店铺名称 + 状态小圆点（绿=启用，灰=停用），点击切换
  - 分割线
  - 操作区：「创建店铺」按钮
  - 分割线
  - 管理区：「店铺管理」「成员管理」按钮
- 交互逻辑：
  - 点击店铺名称切换后刷新为对应店铺数据
  - 点击「店铺管理」在 iframe 内加载店铺管理页
  - 点击「成员管理」在 iframe 内加载成员管理页

### 3.2 成员管理页

#### 3.2.1 成员列表
- 入口：顶部导航栏 → 店铺下拉 → 成员管理
- 页面结构：
  - 页面标题「成员管理(N人)」，N 为当前店铺成员总数
  - 搜索框按姓名/账号ID搜索
  - 表格列：复选框 → 成员信息(头像+姓名) → 账号ID → 手机号 → 组织机构 → 所在店铺 → 加入时间 → 操作
  - 操作按钮：编辑、删除
- 交互逻辑：
  - 点击行进入编辑弹窗
  - 删除操作为软删除，带二次确认
- 账号ID规则：`AC` + 年月 + 5位序号，如 `AC20260700001`，系统自动生成，前端不展示输入框

#### 3.2.2 添加/编辑成员弹窗

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 姓名 | 文本输入 | 是 | 成员真实姓名 |
| 手机号 | 文本输入 | 是 | 11位手机号 |
| 组织机构 | 复选框列表 | 是 | 从当前店铺组织架构中勾选 |
| 账号ID | 系统生成 | - | 后台自动生成，前端不可见、不可修改 |

### 3.3 创建/编辑店铺弹窗

#### 3.3.1 创建店铺
- 入口：店铺列表页「创建店铺」按钮
- 页面结构：模态弹窗，包含以下字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 店铺名称 | 文本输入 | 是 | 最多50字 |
| 店铺域名 | 文本输入 | 是 | 格式 xxx.rebeccashop.com，自动校验唯一性 |
| 默认语言 | 下拉选择 | 是 | 从管理员中台语言字典拉取 |
| 默认币种 | 下拉选择 | 是 | 从管理员中台货币字典拉取 |
| 时区 | 下拉选择 | 是 | 常用时区列表 |

- 交互逻辑：
  - 填写完成后点击「创建」提交
  - 域名输入时实时校验是否已被占用
  - 创建成功后自动跳转到新店铺后台
- 异常处理：
  - 域名已占用时红色提示「该域名已被使用」
  - 必填项为空时按钮置灰不可点击

#### 3.3.2 编辑店铺信息
- 入口：店铺卡片「编辑」按钮
- 页面结构：与创建弹窗相同，预填当前数据
- 交互逻辑：保存后实时更新卡片信息

## 4. 数据指标

### 4.1 核心指标

| 指标名称 | 计算公式 | 数据来源 | 统计周期 | 目标值 |
|----------|----------|----------|----------|--------|
| 店铺创建数 | COUNT(新建店铺) | shop 表 | 日/周/月 | - |
| 活跃店铺数 | COUNT(status=启用) | shop 表 | 日 | - |
| 店铺切换次数 | COUNT(切换事件) | 行为埋点 | 日 | - |

### 4.2 观察指标

| 指标名称 | 计算公式 | 数据来源 | 统计周期 |
|----------|----------|----------|----------|
| 单账号店铺数 | AVG(店铺数/账号) | shop + account 表 | 月 |
| 停用率 | 停用店铺/总店铺 | shop 表 | 月 |
